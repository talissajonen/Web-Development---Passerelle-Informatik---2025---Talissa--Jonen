Flexbox helps arrange elements inside a container. It makes it easy to align and distribute space between items, even when the screen changes.

The container controls layout (display, flex-direction, justify-content, align-items). Items can grow or shrink (flex-grow, flex-shrink).

Container properties:

  • display: flex; → turns on flexbox
  • flex-direction: row; → items in a row
  • justify-content: space-around; → space evenly around items
  • align-items: center; → center items vertically

Item properties:

  • flex-grow: 1; → item can grow
  • flex-shrink: 1; → item can shrink
